## ï¿½ï¸ Installation & Setup

### 1. Repository Setup
Clone the high-performance analytics core:
```bash
git clone https://github.com/shashank-35/House-price-prediction.git
cd House-price-prediction
```

### 2. Dependency Management
Install the analytical environment (Python 3.8+ recommended):
```bash
pip install -r requirements.txt
```

### 3. Model Training
Initialize the machine learning weights:
```bash
python train_model.py
```

### 4. Direct Execution
Launch the premium dashboard experience:
```bash
streamlit run app.py
```
